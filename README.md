## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/RafaelCava/clean-react

# Access
$ cd clean-react

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
